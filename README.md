# MKMAKE

### This script generates a generic makefile directed to a certain programming language

## Supported programming languages:

- Java
- C
- C#
- Nodejs
- Prolog
- Python

## Examples of use

`mkmake java`

`mkmake c`

`mkmake nodejs`

(...)
